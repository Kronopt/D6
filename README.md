# D6
Six sided die (text-based and UI versions)

#### Dependencies
Python 2.7

D6_ui.pyw:
* Module dependencies (can be installed by running `install_dependencies.py`):
  * kivy
  * kivy.deps.glew
  * kivy.deps.sld2


#### How to run
Double click D6.py
* To exit the script, type `q`, `quit` or `exit` and press enter

Double click D6_ui.pyw
